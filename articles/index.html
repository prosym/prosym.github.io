プログラミング・シンポジウムの情報発信・アウトリーチの一環として、各種の寄稿・記事を載せられるスペースを作りました。

プログラミング・シンポジウムや情報科学若手の会に参加したことのある方、ぜひ記事をお寄せください。

寄稿にあたって

  • その時の幹事以外の方から記事をお寄せいただく場合 :
    • 記事は Markdown で記述してください。 (Markdown 以外の形式にしたい場合はご相談ください)
    • 記事はクリエイティブ・コモンズ バージョン 4.0 の「表示—非営利 (CC BY-NC)」でライセンスしてください。
      • 「継承 (SA)」「改変禁止 (ND)」を任意に追加して CC BY-NC-SA または CC BY-NC-ND としていただくこともできます。
    • 記事は基本的にそのまま掲載いたしますが、幹事の判断により、修正をお願いしたり、非掲載とさせていただく可能性があります。
    • 記事内容や著作権者表示を幹事が無断で改変することはありませんが、将来、非掲載になったり、スタイル変更などの影響を受けて外観などが変わったりする可能性はあります。
  • 記事には Canonical link (正規化リンク) を指定できるようになっていますので、他の Web 媒体とクロスポストしていただくことも可能です。 (参考: Google 検索)
